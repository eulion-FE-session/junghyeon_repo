HTML Semantic 태그

header 태그

페이지에 대한 정보를 담는 태그이며 보통 페이지 상단에 위치한다.

nav 태그

다른 페이지나 같은 페이지 안의 다른 부분으로 이동하는 네비게이션 링크를 묶는 태그이다.

홈 | 과제 | 연락처

section 태그

문서나 응용프로그램의 일반적인 섹션을 표현할 때 사용한다.

article 태그

여러가지 아이템들을 묶어 재사용 가능하게 그룹화하는 태그이다.

aside 태그

페이지 전체 내용과 관련성은 있지만 주요 내용과 직접적인 연관은 없는 부가 내용을 넣는 태그이다.

footer 태그

주로 저작권 정보나 서비스 제공자 정보 등을 사이트 하단에 나타낼 때 사용한다.




details / summary 태그

summary 태그 : 클릭해서 내용 보기

details 태그는 추가적인 정보를 나타내거나 사용자가 요청하는 정보를 나타낼 때 사용한다.

summary 태그는 details 요소의 요약이나 캡션 역할을 한다.

Block Element / Inline Element

Block Element

블록 레벨 요소는 부모 요소의 전체 공간을 차지하는 블록을 만든다. 대표적인 태그로는 <h1>~<h6>, <ol>, <ul>, <li>, <p> 등이 있다.

특징 : 세로로 나열된다.

Inline Element

인라인 레벨 요소는 콘텐츠의 흐름을 끊지 않고 요소를 구성하는 태그에 할당된 공간만 차지한다. 대표적인 태그로는 <a>, <em>, <img>, <span> 등이 있다.

특징 : 가로로 나열된다.
